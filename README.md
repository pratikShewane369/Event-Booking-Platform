# Event Booking Platform

A full-stack MERN application where users can browse and book events, while admins manage and confirm bookings.

## Tech Stack

- React (Vite)
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
